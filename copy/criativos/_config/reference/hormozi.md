# Framework de referencia — Hormozi ($100M GOATed Ads)

Terminologia e classificacoes baseadas no framework de Alex Hormozi.

## Estrutura de um ad

Hook -> Body -> CTA

- **Hook**: "Whatever people see and/or hear first." 80% do esforco criativo vai aqui.
- **Body** (Hormozi: "Meat"): "The part of the ad that fulfills your hook."
- **CTA**: "Tell them exactly what to do next. S-P-E-L-L it out."

## Hook types (tipos verbais)

| Tipo | O que faz |
|------|-----------|
| `statement` | Afirmacao direta |
| `question` | Pergunta sim/nao ou aberta |
| `conditional` | Se voce [condicao], entao... |
| `label` | Rotula o publico |
| `command` | Ordem direta |
| `narrative` | Historia/anedota |
| `list` | Lista de passos/itens |
| `exclamation` | Emocao forte |
| `ridiculous-result` | Resultado bizarro/raro |

## Body formats

| Formato | Descricao |
|---------|-----------|
| `demonstration` | Uso ao vivo, comparacoes, antes/depois |
| `testimonial` | UGC, direct-to-camera, walk-n-talk, podcast style |
| `education` | Explainer, how-to, whiteboard, conteudo organico |
| `story` | Storytelling, lifestyle, documentario, skits |
| `faceless` | Screenshots, texto, slides, animacoes |

Bodies atuais: b01-b03 = education, b04 = testimonial, b05 = demonstration.

## Regra 70-20-10 (inovacao criativa)

| Tier | % | O que eh |
|------|---|----------|
| `core` | 70% | Winners comprovados reusados |
| `emerging` | 20% | Modelados de outros nichos |
| `experimental` | 10% | Conceitos novos |

## Volume de producao (benchmark)

- 50 hooks x 3-5 bodies x 1-3 CTAs = 150-750 ads/semana
- 80% do esforco criativo em hooks, 20% em bodies, ~0% em CTAs
